How It Works?

This is a Stopwatch Web Application which is designed in a UserFriendly Manner, for the user to start the stopwatch by clicking on the "Start" button and it will start counting from 00:00:00:00. When the user clicks on the " stop" button it will stop the stopwatch from counting and when the user clicks the "Reset" button it will reset the stopwatch to 00:00:00:00.

To use this App

  • To Start the Stopwatch. Press The Start Button.
  • After pressing the Start button the Timer will Start. To Stop the Timer press the Stop button.
  • To Start from the begining or reset the Timer Press on Reset Button.
